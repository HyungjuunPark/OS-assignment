# 💻 Operating System Concepts Implementation
> **운영체제의 핵심 원리 이해와 프로세스 및 자원 관리 시뮬레이션**

## 📌 개요
- **성격:** 전공 교과 운영체제 실습 과제
- **기술 스택:** C/C++, System API (Process Management)
- **주요 목표:** 프로세스 스케줄링 알고리즘 및 시스템 자원 모니터링 로직 구현

## 🛠 주요 구현 내용

### 1. 시스템 모니터링 및 자원 분석
- **프로세스 추적:** 현재 실행 중인 전체 프로세스 수를 계산하고, 1초마다 실시간으로 현재 시간과 프로세스 현황을 출력하는 모니터링 로직 구현.
- **CPU Load 산출:** 시스템의 CPU 부하율을 계산하여 하드웨어 리소스 사용량 분석.
- **프로세스 정보 상세 출력:** 시스템 내 개별 프로세스의 상세 정보를 추출하여 하위 시스템에 대한 이해도 제고.

### 2. 알고리즘 시뮬레이션 및 데이터 처리
- **SJF(Shortest Job First) 스케줄링:** 가장 짧은 작업 시간을 가진 프로세스를 우선 처리하는 비선점형 스케줄링 시뮬레이터를 구현하여 평균 대기 시간 분석.
- **행렬 데이터 연산 및 파일 입출력:** 행렬 데이터의 연산 결과를 화면에 출력하고 파일(File I/O)로 기록하여 데이터 영속성 관리.
- **기초 연산 구현:** 기본적인 반복문과 입출력을 활용한 구구단 연산 로직 수립.
